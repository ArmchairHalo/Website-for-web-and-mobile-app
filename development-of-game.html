Tools what will be used for the game

For the game i will use a varity of diffrent tools

  • Unity: this will be used to actualy make the diffrent levels and scenes for the game
  • Visual Studio: this will be used to write code for the game
  • Piskel: this will be used for creating pixel art character sprites and might also be used for backgrounds
  • GitHub: this will be used so im able to keep backups of the files and also be used so that im able to work from home and at college
  • bfxr: this will be used to make synthetic sound effects
  • Online Sequencer this will be used to make background music
  • Audacity this will be used in to fix or edit the audio what will be used for the game
